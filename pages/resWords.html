Palabras Reservadas

Son aquellas que se usan para construir sentencias de JavaScript. NO pueden ser usadas como nombres de variables, funciones o identificadores

Control de flujo y estructuras de control

  • break // Salir de un bucle o switch
  • case // Definir un caso dentro de un switch
  • catch // Capturar errores en try...catch
  • continue // Saltear una iteracion en un bucle
  • default // Caso por defecto en switch
  • do // Ejecutar un bloque al menos una vez en do...while
  • else // Alternativa en una estructura if
  • finally // Bloque que siempre se ejecuta en try...catch
  • for // Bucle for
  • if // Condicional if
  • return // Devolver un valor en una funcion
  • switch // Estructura de seleccion multiple
  • throw // Lanzar un error
  • try // Manejar excepciones
  • while // Bucle while

Declaración de variables y funciones

  • var // Declarar una variable (NO USAR O MUEREN GATITOS)
  • let // Declarar una variable
  • const // Declarar una constante
  • function // Declarar una funcion

Manejo de objetos y clases

  • class // Definir una clase
  • extends // Heredar de otra clase
  • super // Referenciar la clase de padre en una subclase
  • this // Referenciar el objeto actual
  • new // Crear una instancia de un objeto o clase
  • delete // Eliminar propiedades de un objeto

Módulos (ES6)

  • export // Exportar funciones, clases o variables de un módulo
  • import // Importar funciones, clases o variables de un módulo

Valores booleanos

  • true // Valor booleano verdadero
  • false // Valor booleano falso
  • null // Valor nulo (ausencia intencionada de valor)
  • undefined // Valor no definido

Operadores y manipulación de datos

  • typeof // Obtener el tipo de una variable
  • instanceof // Verificar si un objeto pertenece a una clase
  • in // Verificar si una propiedad existe en un objeto

Palabras clave de Generators y Asincronía

  • async // Declarar una funcion asincrona
  • await // Esperar el resultado de una funcion asincrona
  • yield // Pausar y reanudar un generator
